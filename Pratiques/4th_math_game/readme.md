Ce code est un petit **quiz de calcul mental**.

En résumé :
1.  Il pose une série de questions aléatoires (soit des **additions**, soit des **multiplications**).
2.  Il vérifie si la réponse de l'utilisateur est juste ou fausse.
3.  À la fin, il affiche le **score total** et un petit commentaire sur la performance (exemple : "EXCELLENT", "Pas mal !", etc.).
